`rebar3` custom templates for escript.

### Usage
```
$ rebar3 new script <name>
```

This creates a new directory `<name>` with the following directory structure
```

.
├── LICENSE
├── Makefile
├── README.md
├── elvis.config
├── rebar.config
└── src
    ├── <name>.app.src
    └── <name>.erl

1 directory, 7 files
```
